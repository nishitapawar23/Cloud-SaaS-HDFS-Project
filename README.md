# Cloud-SaaS-HDFS-Project
Mini cloud-based SaaS system using HDFS with distributed file storage and basic encryption concepts
## Architecture
This project simulates a cloud environment over LAN using HDFS where files are stored in distributed blocks across nodes.

## Key Concepts
- Distributed Storage (HDFS)
- Data Blocks
- Fault Tolerance
- Basic Security (Encryption concept)

## Future Scope
